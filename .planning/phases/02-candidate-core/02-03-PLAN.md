  - Color dot (green for created/hired, blue for status_change, purple for tier_change, etc.)
  - Event description: "Status changed from [From] to [To]" or "Tier changed from [From] to [To]" or "Candidate created"
  - Use STATUS_LABELS and TIER_LABELS to format the from/to values

- Style: thin gray vertical line connecting dots, compact spacing

**candidate-drawer.tsx — CandidateDrawer component:**
- Props: candidateId (string | null), onClose (() => void)
- Uses Sheet from @/components/ui/sheet with side="right" on desktop, side="bottom" on mobile
- Use useIsMobile from @/hooks/use-mobile.ts for responsive behavior
- Width: className="w-full sm:max-w-[480px]" per CONTEXT.md (400-500px)
- Sheet open state controlled by !!candidateId

CRITICAL: When candidateId is provided, fetch candidate + events using the `fetchCandidateProfile` SERVER ACTION from `@/lib/actions/candidates`.    - Portfolio URL: clickable link, opens in new tab

   - Section header "History"
